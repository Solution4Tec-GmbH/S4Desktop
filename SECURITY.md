# Security Policy – Solution4Tec Desktop

## Supported Versions

We generally support security updates for the **current major version** of the Solution4Tec desktop application.

Security‑relevant fixes are delivered via the built‑in auto‑update mechanism whenever possible. We recommend that all users enable updates or regularly check for updates from within the application.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in the Solution4Tec desktop app or related infrastructure, please report it **privately** to us.

- **Email:** [security@solution4tec.com](mailto:security@solution4tec.com)
- **Subject:** `SECURITY: <short description>`

Please include, where possible:

- A clear description of the issue and potential impact.
- Steps to reproduce or proof‑of‑concept code.
- Any logs, screenshots, or network traces that help us understand the issue.
- Your contact information for follow‑up.

### Responsible Disclosure Guidelines

We kindly ask that you:

- Do **not** publicly disclose the vulnerability until we have had a reasonable time to investigate and release a fix.
- Avoid accessing or modifying data that does not belong to you.
- Avoid actions that could degrade the availability or integrity of our services (e.g., denial‑of‑service testing).

We are committed to:

- Acknowledging receipt of your report.
- Providing you with updates as we triage and address the issue.
- Not taking legal action against good‑faith security researchers who follow these guidelines.

## Data Handled by the Desktop App

The Solution4Tec desktop app is a client application that:

- Connects to Solution4Tec backend services and company instances.
- May store local configuration, application settings, and cached data on the device.
- Does **not** itself provide hosting or backend data storage.

The exact data processed depends on your organization’s Solution4Tec configuration and connected services.

For details about data protection and privacy, please refer to the privacy and data‑processing documentation provided by Solution4Tec.

## Updating Securely

The desktop app uses an integrated update mechanism to check for and download new versions from Solution4Tec’s official release channels.

To reduce risk:

- Only download installers and updates from official Solution4Tec sources.
- Do not run unofficial or modified builds of the application.
- Keep your operating system and antivirus software up to date.

If you suspect that an installer or update has been tampered with, **do not install it** and contact [security@solution4tec.com](mailto:security@solution4tec.com) immediately.
